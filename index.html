Learn Git - Version Control

Basic Commands

COMMAND DESCRIPTION Example
git clone For making a copy of directory from github to your website git clone https://github.com/yashraj1309/learngit.git
git add file-name For add specific for commit when you commit git add index.html
git commit -m "message" For commit changes you made on file. Simply you save the changes git commit -m "Updating Index.html"
git status To see the status of repository : what is going on git status - you will see message like below 
                        On branch main
                        Changes to be committed:
                        (use "git rm --cached ..." to unstage)
                                new file:   index.html
                        Changes not staged for commit:
                        (use "git add ..." to update what will be committed)
                        (use "git restore ..." to discard changes in working directory)
                                modified:   index.html
git push when you made change on cloned repository github does not effect the original one from which you cloned it. For changing that you need to use push command. git push - you will see message like below. 
                        Enumerating objects: 3, done.
                        Counting objects: 100% (3/3), done.
                        Delta compression using up to 2 threads
                        Writing objects: 100% (3/3), 676 bytes | 8.00 KiB/s, done.
                        To https://github.com/yashraj1309/learngit.git
                        * [new branch]      main -> main
git pull Mainly to access the latest version of your code: When two different developers working on the same project same time and same feature then it will very usefull git pull https://github.com/yashraj1309/learngit.git 
                        Updating 26a9cc5..2763abc
                        Fast-forward
                        index.html | 1 +
                        1 file changed, 1 insertion(+)
                        On branch main

                        nothing to commit, working tree clean
git log for current work / chages / updates made on repository git log - you will see log like below 
                        commit c28cbc30838da1e5b1bdf71aee39583929 
                        Author: github profile name 
                        Date:   Sat Jul 30 22:53:26 2022 +0530

                        Updating Index.html
git reset & git reset --hard For undo the chages. If you want to remove all the changes from initial then you need to use git reset -hard command git reset / git reset --hard / git reset --hard origin/master
